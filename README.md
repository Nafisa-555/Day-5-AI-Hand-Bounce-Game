# 🎮 AI Hand Gesture Ball Catch Game

This project is part of my **5 Days OpenCV Challenge**, where I explored Computer Vision by building small and fun projects.

## 💡 Project Idea

This is an interactive **Hand Gesture Game** where:

- You control a **bucket using your hand (thumb + index finger)**
- Multiple balls move and bounce on the screen
- You can **catch/bounce balls using your hand**
- Score increases based on interaction

---

## ✨ Features

- Real-time hand tracking using MediaPipe  
- Gesture-based control (thumb + index finger)  
- Multiple bouncing balls  
- Glow effect for better visuals  
- Real-time score display  
- Smooth hand movement  

---

## 🛠️ Technologies Used

- Python  
- OpenCV  
- MediaPipe  

---
## 📚 What I Learned

- Hand tracking using MediaPipe  
- Real-time interaction with OpenCV

- Improving UI with simple effects
